# IoT-Smart-Irrigation-Dashboard

📌 Overview

This project is a smart irrigation dashboard that shows real-time data from field sensors. It helps in monitoring environmental conditions and controlling water usage efficiently.

🎯 Purpose
Save water
Reduce manual work
Monitor field conditions remotely

⚙️ What I Did
Built a web dashboard using HTML, CSS, and JavaScript
Connected to ThingSpeak cloud to get live data
Displayed temperature, humidity, soil moisture, and water level
Implemented automatic refresh every 5 seconds
Showed pump ON/OFF status dynamically

🛠️ Technologies
HTML, CSS, JavaScript
ThingSpeak Cloud

Conclusion: 
This project shows how IoT can be used for smart irrigation to improve efficiency and reduce water wastage.
